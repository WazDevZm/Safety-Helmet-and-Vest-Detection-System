## Image Specifications:
- Hero images: 1200x800px minimum
- Logo images: 200x200px minimum
- Card images: 400x300px minimum
- All images should be optimized for web (compressed)
- Use WebP format when possible for better performance

## Color Guidelines:
Images should complement the AYM color scheme:
- Adventurer Blue: #2E5BBA
- Pathfinder Red: #DC143C
- Pathfinder Yellow: #FFD700
- AYM Green: #228B22
- PCM Blue: #4169E1
